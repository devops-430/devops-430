Introduction to DevOps

Course Overview

Prerequisites: Basic computer literacy, familiarity with command line interface

Duration: 4 weeks (20 hours total)

Learn the fundamentals of DevOps practices, tools, and culture.

Module 2: Version Control with Git

  • Working with remote repositories

Module 3: Continuous Integration

  • Introduction to CI/CD
  • Jenkins fundamentals
  • Building pipelines
  • Automated testing

Module 4: Infrastructure as Code

  • Introduction to IaC
  • Basic Terraform concepts
  • Cloud provisioning basics
  • Configuration management overview

Module 5: Monitoring & Logging

  • Importance of monitoring
  • Basic Prometheus concepts
  • Log management basics
  • Alerting fundamentals

Hands-on Labs

  • Lab 1: Set up your first CI pipeline
  • Lab 2: Infrastructure provisioning exercise
  • Lab 3: Monitoring setup simulation
  • Lab 4: Collaborative Git workflow

Course Requirements

  • Access to a computer with internet connection
  • Basic understanding of Linux commands
  • Familiarity with any programming/scripting language
  • GitHub account (free tier)
